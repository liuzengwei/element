# Element UI Test Project

基于 xianniu-delivery-cloud-web 依赖配置的测试项目

## 技术栈

- Vue 2.7.14
- @liuzengwei/element-ui ^2.15.5-xn.13
- Vue Router 3.0.2
- Vuex 3.1.0
- Axios 0.18.1
- Dayjs 1.11.10
- Lodash 4.17.21
- ECharts 5.5.1

## 项目设置

```bash
npm install
```

## 开发运行

```bash
npm run dev
# 或
npm run serve
```

## 构建打包

```bash
# 生产环境
npm run build

# 测试环境
npm run build:test
```

## 代码检查

```bash
npm run lint
```
